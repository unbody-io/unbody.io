# AudioFile

_AudioFile represents audio content, providing a link to the file alongside important metadata._

| Field        | Type   | Description                               |
| ------------ | ------ | ----------------------------------------- |
| \_\_typename | string | Indicates the type of the document.       |
| duration     | number | Duration of the audio in seconds.         |
| ext          | string | File extension of the audio file.         |
| mimeType     | string | MIME type of the audio file.              |
| order        | number | Order or sequence of the audio file.      |
| originalName | string | Original name of the audio file.          |
| remoteId     | string | Remote identifier of the audio file.      |
| size         | number | Size of the audio file in bytes.          |
| sourceId     | string | Source identifier of the audio file.      |
| url          | string | URL to access or download the audio file. |
